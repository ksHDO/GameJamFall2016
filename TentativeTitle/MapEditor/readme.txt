CTRL + T -> Switch Textures loaded in the Content pipeline
CTRL + A -> Change map dimensions
CTRL + O -> Open a map
CTRL + S -> Save a map
CTRL + N -> Create a new map
WASD -> Move camera
TILDE or Left-Click -> PlaceTile
R -> Reset Zoom & Camera Position
P -> Show Collision
O -> Show Triggers (I don't think this is incorporated)
Q -> Decrement selected tile ID
E -> Increment selected tile ID

Right-Click -> Place Tile of ID 0.
